# FlexMod 用户使用手册

## 软件简介

FlexMod 是一款专为《七日杀》(7 Days to Die) 游戏设计的模组管理工具，具有双重功能定位：

1. **面向模组作者**：提供编辑器工具，为游戏模组添加可配置选项
2. **面向普通玩家**：提供玩家模式，根据个人喜好调整模组功能

通过 FlexMod，模组作者可以为自己的模组创建灵活的配置选项，而玩家则可以通过简单的界面调整这些选项，无需修改模组文件。

### 主要功能

- **模组目录管理**：配置游戏模组目录路径
- **FlexMod 管理**：添加、查看已有的 FlexMod
- **FlexMod 编辑器**：（面向模组作者）创建和编辑 FlexMod 配置
- **玩家模式**：（面向普通玩家）实时调整和应用 FlexMod 设置
- **多语言支持**：支持英文和中文界面

## 安装指南

### 系统要求

- Windows 操作系统
- 《七日杀》游戏已安装

### 安装步骤

1. **下载 FlexMod**：获取 FlexMod 软件包
2. **运行软件**：双击 `FlexMod.exe` 文件启动程序

## 首次使用配置

当您首次运行 FlexMod 时，系统会提示您设置模组目录路径。

1. **选择模组目录**：点击浏览按钮，导航到您的《七日杀》游戏目录下的 `Mods` 文件夹
   - 例如：`C:\Program Files (x86)\Steam\steamapps\common\7 Days To Die\Mods`

2. **验证路径**：系统会自动验证所选路径是否为正确的模组目录

3. **保存设置**：验证通过后，设置会自动保存

4. **进入主页**：设置完成后，系统会自动切换到主页

## 主界面介绍

FlexMod 主界面包含以下几个主要部分：

### 导航栏

位于界面顶部，包含以下按钮：

- **Home/主页**：切换到主页，管理 FlexMod
- **Setting/设置**：进入设置页面，配置模组目录
- **Language/语言**：切换界面语言（英文/中文）

### 主页

主页分为左右两栏：

#### 左侧：FlexMod 列表

- **FlexMod 列表**：显示已添加的 FlexMod
- **刷新按钮**：刷新 FlexMod 列表
- **+ FlexMod 按钮**：添加新的 FlexMod
- **玩家模式按钮**：切换玩家模式

#### 右侧：玩家页面

仅在玩家模式启用时显示，用于实时调整和应用 FlexMod 设置。

### 设置页面

- **模组目录输入框**：显示当前配置的模组目录路径
- **重新设置按钮**：重新选择模组目录
- **定位文件夹按钮**：在资源管理器中打开当前模组目录
- **说明文本**：显示模组目录设置的相关说明

## 使用指南

### 模组作者使用指南

#### 添加 FlexMod

1. **点击主页上的 " + FlexMod " 按钮**

2. **选择模组**：在弹出的对话框中，从列表中选择要添加 FlexMod 功能的模组

3. **确认添加**：点击 "OK" 按钮确认添加

4. **完成**：系统会自动创建 FlexMod 所需的文件结构，并将模组添加到列表中

#### 编辑 FlexMod

1. **在主页的 FlexMod 列表中双击**要编辑的 FlexMod

2. **使用编辑器**：系统会打开 FlexMod 编辑器窗口，您可以在此：
   - 添加、编辑功能块
   - 设置功能块参数
   - 配置执行代码
   - 调整分组设置

3. **保存更改**：编辑器会自动保存您的更改

### 普通玩家使用指南

#### 使用玩家模式

1. **在主页上点击玩家模式按钮**（图标按钮）

2. **选择 FlexMod**：在左侧列表中选择一个 FlexMod

3. **调整设置**：在右侧玩家页面中调整 FlexMod 的各项设置

4. **应用更改**：调整完成后，设置会自动应用

### 通用操作

#### 切换语言

1. **点击导航栏上的 "Language/语言" 按钮**

2. **语言切换**：界面语言会在英文和中文之间切换

## FlexMod 编辑器使用指南

FlexMod 编辑器是**面向模组作者**的核心工具，用于创建和编辑 FlexMod 配置，具有以下功能：

### 功能块类型

- **开关 (Boolean)**：创建开/关类型的选项
- **下拉选项 (Dropdown)**：创建多选项的下拉菜单
- **整数滑块 (Integer Slider)**：创建整数范围的滑块控制
- **浮点滑块 (Float Slider)**：创建浮点数值范围的滑块控制

### 编辑功能块

1. **添加功能块**：点击 "添加功能块" 按钮，选择功能块类型

2. **设置参数**：
   - **唯一标识 (Unique ID)**：功能的唯一标识符
   - **显示名称 (Display Name)**：在游戏中显示的名称
   - **描述 (Description)**：功能的详细说明
   - **分组 (Group)**：功能所属的分组
   - **默认值 (Default Value)**：功能的默认值

3. **配置执行代码**：
   - **开启时执行代码**：功能启用时执行的代码
   - **关闭时执行代码**：功能禁用时执行的代码

4. **保存配置**：编辑器会自动保存您的更改
